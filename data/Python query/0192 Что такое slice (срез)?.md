Что такое slice (срез)?

[answer_content]
slice (срез) — это метод, который позволяет нам получить только часть списка, кортежа или строки. Для этого мы используем оператор среза [ ].

```py
     (1,2,3,4,5)[2:4]
       # (3, 4)

     [7,6,8,5,9][2:]
        #[8, 5, 9]

     'Hello'[:-1]
       # 'Hell'
```
